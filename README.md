# SmartDesk AI

Aplicação distribuída AI-powered para triagem inteligente de chamados de suporte técnico.

Stack principal:
- Java 25
- Spring Boot 4
- Spring Framework 7
- Spring Cloud 2025.1.x
- Spring AI 2.x
- Resilience4j
- Eureka
- Config Server
- Spring Cloud Gateway
- GraphQL
- Spring Cloud Function
- Prometheus
- Grafana
- JMeter
